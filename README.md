## ‚ú® Features

- **Real-time Weather:** Fetches live data including temperature, humidity, and wind speed.
- **Dynamic Backgrounds:** Background gradients change dynamically based on weather conditions (Clear, Clouds, Rain).
- **Glassmorphism Design:** Modern UI with blurred transparency and soft shadows.
- **Responsive Layout:** Fully optimized for Mobile, Tablet, and Desktop using CSS Flexbox and Media Queries.
- **Interactive UI:** Smooth loading spinners and custom error handling for "City Not Found" scenarios.
- **Component-Based:** Organized structure with separate components for `WeatherCard` and `ErrorMessage`.
